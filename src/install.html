インストール

Vue.jsではIE8はサポートしてないのでIE8のサポートが必要な場合は使用することが出来ません。

そのまま使用する場合

公式サイトからダウンロードしたファイルをそのままscriptタグで読み込むと、Vueというグローバル変数がexportされます。

また、cdnjsにもホスティングされていますし、bowerを使用することも出来ます。

モジュールビルドライブラリを使用する場合

Vue.jsがビルドで使用しているcomponentやbrowserifyを使ってrequireで読み込む方法もサポートされています。

var Vue = require('vue');
