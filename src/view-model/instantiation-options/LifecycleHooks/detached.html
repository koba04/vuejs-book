detached

vm.$elがdirectiveやvm.$appendTo()によってDOMツリーから削除された時に呼ばれます。

vm.$elを直接操作して削除された場合はこのフックは呼ばれません。


//<div id="main"></div>

vm = new Vue({
  el: "#main",
  template: "{{name}}",
  data: {
    name: "1st"
  },
  detached: function() {
    console.log("detached");
    console.log(document.body.innerHTML); // ""
  }
});

vm.$remove();
