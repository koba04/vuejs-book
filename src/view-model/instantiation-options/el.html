el

ViewModelに紐付けたいDOM要素をquerySelectorの書式で指定しています。

指定したDOM要素はvm.$elのように$elでアクセスすることが出来ます。

もしこのオプションが指定されていない場合は、DOM要素が新しく作成されます。

(tagNameが指定されていればそのtagが、なければdivで作成されます)

vm = new Vue({
  el: "#main"
});

<div id="main"></div>
